# 📘 Actividad 7 - Taller: Vectores y Matrices

Repositorio desarrollado para la **Actividad 7: Taller de Vectores y Matrices**, en el cual se solucionan cinco ejercicios mediante algoritmos implementados en **Python**.

El objetivo principal es aplicar conceptos de programación, arreglos, vectores, matrices, funciones, clases y estructuras de datos, siguiendo buenas prácticas de codificación y realizando pruebas de escritorio para validar los resultados.

---

## 🎯 Objetivo del proyecto

Resolver los problemas propuestos en el taller por medio de algoritmos en Python, teniendo en cuenta:

- Definición clara de variables.
- Uso de funciones para organizar el código.
- Aplicación de listas, arreglos y matrices.
- Implementación de clases cuando es necesario.
- Buenas prácticas de programación.
- Pruebas de escritorio para comprobar el funcionamiento.
- Validación de casos básicos y posibles errores.

---

## 🧩 Ejercicios desarrollados

### 1. Ordenamiento burbuja

Se genera un arreglo de **N elementos** y se ordena de **mayor a menor** utilizando el método burbuja.

**Resultado esperado:**

```text
Arreglo original: [8, 3, 10, 1, 7]
Arreglo ordenado: [10, 8, 7, 3, 1]
```

---

### 2. Determinante de una matriz NxN

Se implementa un algoritmo para calcular el determinante de una matriz cuadrada.  
La matriz cuenta con el método `fill_matrix()`, el cual permite llenarla con números aleatorios entre **1 y 20**.

**Ejemplo:**

```text
Matriz:
[2, 3]
[1, 4]

Determinante: 5
```

---

### 3. Examen de admisión

Se simula un examen de admisión compuesto por dos pruebas:

- Aptitud matemática.
- Aptitud verbal.

El programa calcula:

- Puntaje obtenido por cada estudiante en cada examen.
- Puntaje total por estudiante.
- Promedio de cada examen.
- Promedio total.
- Estudiantes con puntaje superior o igual al promedio.
- Mayor puntaje y credencial del estudiante que lo obtuvo.

---

### 4. Matriz de ventas

Se crea una matriz de ventas de **N vendedores** durante **M años** de operación.

El algoritmo calcula:

- Total de ventas por cada vendedor.
- Total de ventas por cada año.
- Gran total de ventas de la empresa.

**Ejemplo:**

```text
Total por vendedor: [6000, 7500, 9000]
Total por año: [4500, 7500, 10500]
Gran total de ventas: 22500
```

---

### 5. Tabla de clasificación de fútbol

Se actualiza la tabla de clasificación de un campeonato profesional de fútbol después de una fecha jugada.

El programa calcula:

- Partidos jugados.
- Partidos ganados.
- Partidos empatados.
- Partidos perdidos.
- Goles a favor.
- Goles en contra.
- Diferencia de gol.
- Puntos.
- Tabla ordenada de acuerdo con la puntuación.
